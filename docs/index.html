Functions

get_unicode_version ()
Return version of the Unicode database.

Returns:

    version string, e.g. 8.0.0
get_bidi_class (code)
Get bidirectional class of a codepoint.

Parameters:

  • code Unicode codepoint

Returns:

    value according to ucdn.UCDN_BIDI_CLASS_* and as defined in UAX#44.
paired_bracket (code)
Get paired bracket for a codepoint.

Parameters:

  • code Unicode codepoint

Returns:

    paired bracket codepoint or the original codepoint if no paired bracket character exists
paired_bracket_type (code)
Get paired bracket type for a codepoint.

Parameters:

  • code Unicode codepoint

Returns:

    value according to ucdn.UCDN_BIDI_PAIRED_BRACKET_TYPE_* and as defined in UAX#9.
compat_decompose (code)
Get compatibility decomposition for a codepoint.

Parameters:

  • code Unicode codepoint

Returns:

    table containing the decomposed codepoints.
